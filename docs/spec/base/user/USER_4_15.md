## End画面

- 目的・用途

本画面は、ワークフローのアクションがすべて完了した際もしくはアクションと途中に強制終了した際に表示される画面である。

- 利用方法

フローで設定されたアクションをすべて行う。または、アクションの途中に強制終了を行う。

- 機能内容

  - 利用申請/利用登録ワークフローの場合、以下のメッセージを表示する。

    日本語：「申請が承認されました  
　　　 アイテム詳細ページからコンテンツをダウンロードしてください」

    英語：「Application approved.  
　　　 Please download the content from the item details」

    アイテム詳細ページ（items details）押下で、申請先アイテムの詳細画面へ遷移する。

  - 利用申請/利用登録ワークフロー以外の場合は、メッセージは表示されない。
