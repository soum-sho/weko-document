## 永続識別子

エンドポイント：/admin/persistentidentifier/

表内のいずれかの○に合致すれば、永続識別子ページの閲覧およびレコードの閲覧が出来ます。

|ロール|システム<br>管理者|リポジトリ<br>管理者|コミュニティ<br>管理者|登録ユーザー|一般ユーザー|ゲスト<br>(未ログイン)|
|---|---|---|---|---|---|---|
|利用可否|○|×|×|×|×|×|
