### アクティビティタブ

タブはToDoタブ、Waitタブ、All（全て）タブの3種類です。

いずれかの○に合致すればタブにアクティビティが表示されます。

また、表示内容はユーザーのロールとアクティビティの状態（作成中・編集中・承認待ちなど）によって動的に変化します。
