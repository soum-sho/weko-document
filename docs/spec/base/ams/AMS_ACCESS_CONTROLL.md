## 未病フロント アクセスコントロール

未病データベースにおけるアクセスコントロールについて記載する。

### 用語説明

本書では以下の用語で統一する。

| 用語       | 説明                                                  |
| ---------- | ----------------------------------------------------- |
| フロント   | 未病データベースのフロントエンド                      |
| WEKO       | 未病データベース用の WEKO3 リポジトリ（バックエンド） |
| 認証ユーザ | 目標 2 グループ所属のユーザ                           |
